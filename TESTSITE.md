# Publiceren als GitHub Pages-testsite

De map is gereed voor publicatie als statische testsite.

1. Maak onder GitHub-account `MiekeDams` een lege repository `zinergi-testsite`.
2. Voeg geen README, `.gitignore` of licentie toe.
3. Plaats alle bestanden uit deze map in de hoofdmap van de repository.
4. Kies in GitHub: **Settings → Pages → Deploy from a branch → main / root**.
5. De verwachte test-URL wordt `https://miekedams.github.io/zinergi-testsite/`.

De testsite verwerkt geen betalingen. De afrekenknop verwijst naar de bestaande beveiligde Zinergi-checkout.
