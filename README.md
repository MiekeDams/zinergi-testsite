# Zinergi standalone website

Deze statische versie draait volledig zelfstandig en is voorbereid op latere integratie.

## Lokaal starten

Open `index.html` direct in een browser, of serveer de map via een eenvoudige lokale webserver.

## Integratiepunten

- Alle primaire shop- en agendakoppelingen verwijzen nu naar `https://zinergi.nl`.
- Contact gebruikt directe `mailto:`- en `tel:`-links.
- De site gebruikt alleen lokale HTML, CSS, JavaScript en één lokale heroafbeelding.
- De merk-wisselaar koppelt Zinergi aan Avena Aalsmeer en Avena Lombok, terwijl ieder domein technisch zelfstandig blijft.
- Voor integratie in Shopify kunnen de secties uit `index.html` één voor één naar Liquid-secties worden vertaald.
- Kleuren en typografie staan als centrale variabelen bovenaan `styles.css`.

## Referentierichting

Japandi: warm kalkwit, donker grijsgroen, taupe, verfijnde serif-titels, dunne lijnen, rustige beweging en veel witruimte. De ontwerprichting is afgeleid van zowel Avena Living Aalsmeer (`avenaliving.nl`) als Avena Living Lombok (`lombok.avenaliving.com`). De vormgeving is eigen aan Zinergi en neemt geen Avena-assets of code over.
